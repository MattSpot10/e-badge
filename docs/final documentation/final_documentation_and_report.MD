## Project Repository Overview  
This repository contains all design files, documentation, and resources for the BYU-Idaho Dev Board Badge senior project. The board is a wearable, student-focused development platform designed for educational use across the Electrical and Computer Engineering and Computer Science programs.

### Key Features  
- Wearable Minibadge-compatible form factor  
- Support for both C and MicroPython environments  
- Designed for multi-semester reuse  
- Educational ecosystem compatibility (Grove, STEMMA, SAO)  
- Rechargeable battery support with USB-C charging  
- Standards-based I²C, SPI, and USB interfaces
